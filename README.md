# 通配符
1. *  0,1,more char 
2. ?  1 char
3. [character] one character in set
4. [!character] no one character in set
5. [[:class:]] one char in class
[:digit:] number
[:lower:] lower char 
[:upper:] upper char 
[:alnum:] num + alpha
[:alpha:] lower + upper
